## Installation
> Install npm packages
```
npm install
```

### Clone
- Clone this repo to your local machine using `https://github.com/gurveerdhindsa/nimbus.git`

### Development
```
npm run build
```
```
npm run dev
```
> To start webpack-dev-server. Electron will launch automatically
